# Contributing

Thanks for your interest in contributing.

## Development setup

1. Install dependencies:

```bash
npm install
```

2. Create local environment file from template:

```bash
cp .env.example .env
```

3. Start dev server:

```bash
npm run dev
```

## Pull request guidelines

1. Keep pull requests focused and small.
2. Use clear commit messages.
3. Include steps to test your change.
4. Do not include secrets, credentials, or `.env` files.

## Reporting issues

When opening an issue, include:

1. What happened.
2. Expected behavior.
3. Steps to reproduce.
4. Screenshots/logs if relevant.
